# DFT IW Stimulation – Masticatory Function (Data & Code)

This repository contains **de-identified** data and code for the randomized crossover study:

*Interferential current stimulation during dental facial treatment improves masticatory function*.

## Contents
- `data/processed/DFT_glucose.csv` — glucose elution data (mg/dL), long format with `Time` = Pre/Post.
- `data/processed/DFT_VAS.csv` — VAS data (mm), long format with `Time` = Pre/Post.
- `code/01_analysis.R` — primary and sensitivity analyses used in the manuscript.
- `code/glucose_test.R`, `code/VAS_plot.R` — exploratory scripts (kept for transparency).
- `results/` — model summaries and figures generated by `code/01_analysis.R`.
- `DATA_DICTIONARY.md` — variables and units.
- `CITATION.cff` — citation metadata.
- `LICENSE-CODE` (MIT) / `LICENSE-DATA` (CC BY 4.0).

## Reproducibility
1. Install R (≥ 4.3) and run `code/requirements.R` to install packages.
2. From the repository root, run: `Rscript code/01_analysis.R`  
   Outputs will be written to `results/`.
3. All random processes are seeded (`set.seed(20250824)`).

## Ethical note
All datasets are de-identified according to IRB approval and consent; no PII/PHI are included.
